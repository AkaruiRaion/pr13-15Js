HTML (HyperText Markup Language) — язык разметки веб-страниц. Он используется для создания структуры веб-документа и определения его содержимого.
CSS (Cascading Style Sheets) — каскадные таблицы стилей. Этот язык используется для управления внешним видом веб-сайта, включая шрифты, цвета, отступы, рамки и многое другое.
JavaScript — скриптовый язык программирования, который используется для добавления интерактивности на веб-страницы. Он позволяет создавать динамические элементы, обрабатывать события, выполнять анимации и взаимодействовать с сервером через AJAX.
PHP (Hypertext Preprocessor) — скриптовой язык программирования общего назначения, чаще всего используемый для разработки веб-приложений. Он может быть использован для обработки данных на стороне сервера, создания динамических страниц и взаимодействия с базами данных.
